### ② 사용자 직접 조치 — Firestore 콘솔에 joinCodes 문서 만들기

세션 A가 규칙과 코드를 고친 뒤, Firestore 콘솔에서 기존 학급마다 조회 문서를 하나씩 만들어야 합니다. 이걸 안 만들면 **학생이 입장할 수 없습니다.**

1. Firebase 콘솔 → Firestore Database
2. `joinCodes` 컬렉션 생성
3. 문서 ID를 **입장 코드 그대로** (예: `TEST-1`)
4. 필드 하나만: `classId` (문자열) = 해당 학급의 문서 ID

앞으로 학급을 새로 만들 때도 `classes` 문서와 `joinCodes` 문서를 **쌍으로** 만들어야 합니다(SPEC §5.2).
